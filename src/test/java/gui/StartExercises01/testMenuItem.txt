Задача: при нажатии на соответственные опции в меню будет произведена операция,
определенная в testMenu.

Тест 1:
    При нажатии из меню "Закрытие" опции "Закрыть" будет
        выполнена операция CloseOperation.

Тест 2:
    При нажатии из меню "Тесты" опции "Сообщение в лог" будет
    выполнена операция TestLogOperation.

Тест 3:
    При нажатии из меню "Режим отображения" опции "Системная схема" будет
    выполнена операция SetSystemLookAndFeel.

Тест 4:
    При нажатии из меню "Режим отображения" опции "Универсальная схема" будет
    выполнена операция SetUniversalLookAndFeel.
