Создается окно по центру MainApplicationFrame.
Все значение value - определены в файле resources/localData...
В окне должны быть 2 кнопки: "yesButtonText", "noButtonText".
Функциональность:
1) Действие: нажатие на кнопку "noButtonText"
   Последствия: вылезающее окошко должно быть закрыто; в Logger
   должна быть занесена запись: "No exit"

2)  Действие: нажатие на кнопку закрыть у окна.
    Последствия: вылезающее окошко должно быть закрыто; в Logger
    должна быть занесена запись: "Close exit"

3)  Действие: нажатие на кнопку "yesButtonText"
    Последствия: вылезающее окошко должно быть закрыто; в Logger
    должна быть занесена запись: "Yes exit"; вся программа
    должна быть закрыта.

4) Действие: нажатие кнопки esc.
    Последствия: Должны выполнятся последствия пункта 2).
