Задача: при нажатии сочетаний клавиш будет произведена операция, определенная в testMenu.

Тест 1:
    При нажатии сочетания клавиш Shift+X будет
    выполнена операция CloseOperation.

Тест 2:
    При нажатии сочетания клавищ Shift+T будет
    выполнена операция TestLogOperation.

Тест 3:
    При нажатии из меню "Закрытие" клавиши X будет
    выполнена операция CloseOperation.

Тест 4:
    При нажатии из меню "Тесты" клавиши T будет
    выполнена операция TestLogOperation.

Тест 5:
    При нажатии из меню "Режим отображения" клавиши S будет выполнена операция
    SetSystemLookAndFeel.

Тест 6:
    При нажатии из меню "Режим отображения" клавиши U будет выполнена операция
    SetUniversalLookAndFeel.
